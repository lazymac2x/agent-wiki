# factory/lessons 로그 (append-only)

> [[feedback-loop]]의 제조 적립면. 새 교훈는 **맨 위에 추가**(편집 아닌 append). 시드 항목은 형식 시연용 예시.
> ⚠️ **Reflexion degeneration 경계**: 모든 실패를 규칙으로 굳히면 규칙 폭발 + 과적합 → 일반화 저해.
> 그래서 두 구획을 **물리적으로 분리**한다.

## 분류 규칙 (승격 vs 격리)
- **A 승격(PROMOTE)** 조건: ≥2개 독립 인스턴스에서 재현 ∧ 반례 통과 → 일반원칙. SOP/표준작업/[[qc-gate]]·패턴으로 굳힘.
- **B 격리(ISOLATE)**: 단일 사건은 일반화하지 않고 인스턴스로 보관. 재현되면 A로 승격, 무재현이면 [[pruning]] tombstone.
- 판정 못 하면 B에 둔다(보수적). **단일사건 → 전사 규칙화 = #1 degeneration 모드.**

## A. 검증된 일반원칙 (PROMOTE — 재사용·승격)
> ≥2 인스턴스 재현된 것만. 여기 항목은 SOP/게이트/패턴으로 흡수 대상.

- **L-001** · 2026-06-29 · **문서화됨 ≠ 수행됨.** 합격기준엔 측정치+실행증거를 동봉한다(무측정 PASS=가짜0). build-green≠live-works의 제조판. → [[qc-gate]] _(다수 라인 재현)_
- **L-002** · 2026-06-29 · **체결 불량의 대부분은 토크값이 아니라 순서·균일성.** 손맛은 OPL 1장이 SOP 본문보다 잘 전달 → 핵심포인트는 OPL로 원자화. → [[_EXAMPLE-torque-setting]]
- **L-003** · 2026-06-29 · **외(外)준비 분리(SMED)가 교체시간의 지배변수.** 내준비 미세최적화보다 외준비 선행이 ROI가 크다 → 표준작업 선행조건에 못박는다.

## B. 인스턴스 과적합 격리 (ISOLATE — 단일사건·미일반화)
> 한 설비/공구/로트 고유 사건. **전 라인 규칙화 금지.** 재현 카운트만 적립.

- **I-001** · 2026-06-29 · #3라인 금형B 로케이트링 0.3mm 편마모로 안착불량 1회 · 코드 **DIM**(치수/안착)·근본 5M1E=**Machine**(금형 마모) → [[defect-taxonomy]] · **그 금형 고유**(보전 대상). 전 라인 안착절차 변경 불요. _재현 0회_
- **I-002** · 2026-06-29 · 야간조 토크렌치(자산#TR-07) 교정만료로 +15% 과체결 · 근본 5M1E=**Measurement**(계측기 교정만료) → [[defect-taxonomy]] · **공구 개체문제**(교정주기로 흡수). 작업표준 변경 불요. _재현 0회_

### 적립 운영
- 격리(B) 항목이 **≥2회 재현**되면 A로 승격하며 근거 인스턴스 ID를 함께 적는다.
- A 승격 시 해당 원칙을 SOP/[[qc-gate]]/패턴에 반영하고, 로그엔 포인터만 남긴다(단일 SOT, 복붙 금지).
- 노트가 200줄/25KB 근접하면 오래된 격리 항목을 [[pruning]]으로 tombstone 후 압축.
