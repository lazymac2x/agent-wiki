# OPL: 금형 클램프 볼트 토크 세팅  `[Know-How]`  ⚠️ 예시·현장미검증

> [[opl]] 템플릿 시연용 **예시**. 한 주제 · 5~10분. [[_EXAMPLE-line-changeover]] SOP 4단계를 조립하는 레고블록이다.
> 수치는 설명용 — 실제는 **설비/금형 명판값이 진실**.

**WHY** (왜 이게 중요한가 — 한 줄)
> 과/부족 토크 = 금형 휘·수지 누설·반복 풀림 → 이형불량·버. 값보다 **균일 체결(순서·축력)**이 핵심.

**STEPS / 그림** (비주얼 우선)
1. 토크렌치 클리커를 목표값에 세팅하고 다이얼 잠금
2. 모든 클램프 볼트를 손조임(snug)까지 — 편심 제거
3. **대각선 순서**로 1차 50% → 2차 100% 조임
4. '딸깡' 1회에서 정지 — 추가 회전 금지

```
   1 ─── 3        대각(별표) 순서:
   │     │        50% 한 바퀴 → 100% 한 바퀴
   4 ─── 2        (한쪽 몰아 조이면 금형 들림·편체결)
```

**KEY POINTS** (파라미터·임계·안전 — 수치로)
- 🔑 M16 8.8급 클램프 = **210 N·m ±10%**(180톤 사출기 예시값 · 설비 명판 우선)
- 🔑 2단 조임: **50%(≈105 N·m) → 100%(210 N·m)**, 대각 순서 고정
- 🔑 **마른 나사(무급유) 기준 토크** — 윤활 시 마찰↓ → 같은 토크에 축력 과대 → 명판의 윤활값 적용
- 🔑 체결 후 **페인트 마킹** — 풀림을 육안으로 감지(QC·추적성)
- ⚠️ 클리커 '딸깡' 후 추가 조임 금지(over-torque) · 사용 후 **최저값으로 풀어 보관**(스프링 보호)
- 임계: 정상 **210±21 N·m** / 이탈 **<189**(풀림 위험) · **>231**(나사 항복·소성변형)
